# 实验3： 算术符练习

1 . 对于任意一个二位数N，乘以11的速算算法如下：

**速算法**  
- 将N的十位数作为结果的百位数；  
- 将N的个位数作为结果的个位数；  
- 将N的个位数和十位数相加，作为结果的十位数；如果相加的结果超过10，则进位1加到百位数；
